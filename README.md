# This repository is the experiment code for AISS-VFL

## Experiment

`/config/` Configurations for different tasks.

`/algorithm` Code of used compressor and other algorithms.

`run_*.py` Main entry.

`/train_log` Training log.

`/output` Training result.


## Demo

`python run_vanilla.py`

## Requirement

PyTorch>=2.0
